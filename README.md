# InterviewBot
## Skill Stack
* Language : Python
* FrameWork : Django
* Web : HTML, CSS
* Database : SQLite
* OS : Linux, Windows
* OpenSource : Konlpy, Bootstrap


## Requirement
* Django
* django-crispy-forms
* django-mailgun
* konlpy
* bootstrap

## How to Run
* run saveQuestion.py -> python saveQuestion.py
* runserver -> python manage.py runserver
* If Bot Copy is't run, run ngrok and webhook django with dialogflow
